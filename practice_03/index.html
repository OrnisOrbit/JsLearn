Язык разметки Markdown

Markdown - это язык разметки в котором структура   описывается спец символами.
Эти символы визуально пытаются отразить графическое оформление элементов версткм.

Заголовки

для создания заголовка первого уровня нужно добавить символы знака =
для создания заголовка второго уровня знаки тире -
также можно пользваться # 

            

                    первого уровня 
                    ==============
                    второго уровня
                    --------------
                    ### третьего

Акцент и значимость текста

для указания акцента на слово, фразу или необходимо обернуть текст символами _ или *

Далеко-далеко за словесными горами в стране, гласных и согласных живут рыбные тексты. Инициал составитель lorem текста они что власти напоивший несколько, родного не. Ему несколько однажды предупреждал ipsum последний. По всей, безорфографичный предложения!
