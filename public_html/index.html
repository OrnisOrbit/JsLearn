Markdown

Введение

Markdown - язык разметки, в котором структура строится спец   символами. Эти символы визуально пытаются отразить графическое оформление элементов верстки.

HTML. Взаимодействие с HTML

Заголовки

Для создания заголовка первого уровня в языке Markdown, после заголовка нужно добавить символы равенства =.

Для создания заголовка второго уровня в языке Markdown, после заголовка нужно добавить символы тире -.

Для задачи заголовка любого уровня в языке Markdown можно добавить определенное количество знаков # перед заголовком

Пример:

	Заголовок первого уровня
	========================
	
	Заголовок второго уровня
	------------------------
	
	# Заголовок первого
	## Второго
	###### ...шестого уровня

Приведет к генерации следующего HTML:

Заголовок первого уровня

Заголовок второго уровня

Заголовок первого уровня

Заголовок второго уровня

...шестого уровня

Акцент и значимость текста

для указания акцента на слово, фразу или предложение, необходимо обернуть текст в символы _ или *.

Для указания высокой важности текста, текст следует обернуть в два символа __ или **


	*текст на который установлен акцент*
	_текст на который установлен акцент_
	**текст на который установлен акцент**
	__текст на который установлен акцент__
	

	текст на который установлен акцент
	текст на который установлен акцент
	текст на который установлен акцент
	текст на который установлен акцент

Ссылки

Для создания ссылки в Markdown необходимо использовать следующий синтаксис:


	[пример ссылки](https://example.com)
	[пример ссылки с заголовком](https://example.com "заголовок")
	<https://example.com>
	
Приведет к генерации следующего HTML:

	Пример ссылки
	Пример ссылки с заголовком
	https://example.com

Изображения

Изображения в языке Markdown оформлены как ссылки с дополнительным сигнальным кодом! в самом начале для систем, которые могут содержать Изображения

Пример:

![логотип Markdown](logo.svg)

Списки в Markdown

неупорядоченные списки

Для создания неупорядоченного списка нужно добавить символ *или+или-


	* Первый элемент списка
	* Второй элемент списка
	* Третий элемент списка
	
	+ Первый элемент списка
	+ Второй элемент списка
	+ Третий элемент списка
	
	- Первый элемент списка
	- Второй элемент списка
	- Третий элемент списка

Приведет к генерации следующего HTML:

  • Первый элемент списка
  • Второй элемент списка
  • Третий элемент списка
  • Первый элемент списка
  • Второй элемент списка
  • Третий элемент списка

Упорядоченные списки

для создания упорядоченного списка, перед элементами списка нужно поставить 1, 2, 3...

1. элемент списка
	2. Элемент списка
	3. Элемент списка

Приведет к генерации следующего HTML:

  1. Первый элемент списка
  2. Второй элемент списка
  3. Третий элемент списка
  1. Первый элемент списка
  2. Второй элемент списка
  3. Третий элемент списка

Списки опеределений

для создания списков определений необходимо использовать следующий синтаксис

слово
	: определение слова
	
	другое слово
	: определение другого слова

Приведет к генерации следующего HTML:

Слово:
определение слова
другое слово
определение другого слова

Цитаты

Как в почтовых сообщениях цитирование текста производится символами > в начале каждой строки

Пример:


	>Как в почтовых сообщениях цитирование текста
	>Как в почтовых сообщениях цитирование текста
	>Как в почтовых сообщениях цитирование текста
	>Как в почтовых сообщениях цитирование текста

Таблицы

В различных расширениях языка Markdown, таблицы можно задавать символом |.


	| Клетка заголовок 1 | Клетка заголовок 2 | Клетка заголовок 3 |
	| ------------------ | ------------------ | ------------------ |
	| Клетка 1           | Клетка 2           | Клетка 3           |
	| Клетка 1           | Клетка 2           | Клетка 3           |
	| Клетка 1           | Клетка 2           | Клетка 3           |

Клетка 1 Клетка 2 Клетка 3

Клетка 1 Клетка 2 Клетка 3
